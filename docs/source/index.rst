Django Clear To Build documentation
===================================

Version: |version|

A package for tracking inventory for small-scale electronics manufacturing.
It tracks parts, vendors, orders, inventories (the parts that you have on-hand), projects across several versions (and their bills of material, including cost breakdowns), and finally provides clearance to build---indicating that your inventory of parts is sufficient to complete the project build.
Progressively tracks inventory across multiple pending builds to ensure that each build has all parts available.
